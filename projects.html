React Dictionary

Allows user to search a word and the provides definition, pronunciation, examples, synonyms, and images or the word. Created using React, Bootstrap, and REST API to provide data.

React Weather

Provides real-time weather data and 6 day future forecast for searched location. Displays expected weather conditions with animated weather icons. Created using React, Bootstrap, and REST API to provide data.

Battery Indicator

Simple page that displays users current battery level with percentage and colored level indicator. Also allows user to toggle between light and dark mode. Created using Vanilla JavaScript

Cryptocurrency Tracker

Page that provides current market data for cryptocurrency values. Market changes are highlighted in either red or green depending on postive or negative changes. Also user can search for a specific currency using search bar. Created with Vanilla JavaScript and REST API to provide data.

Vanilla Weather

Weather application that utilizes geolocation to update weather in user's current location or allows user to search locations and recieve up-to-date weather conditions. Provides forecast in 3-hour intervals. Created using Vanilla JavaScript, Bootstrap, and REST API to provide data.

Color Generator

Simple page that randomly generates hex color values and updates background color as user presses button. Page also has a simple setting that updates the value and background color with pre-loaded values. Created with Vanilla JavaScript.
